# PhishGuard Pro User Manual

## Getting Started

### Installation
1. Download the PhishGuard Pro extension
2. Open Chrome and navigate to `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked" and select the extension folder
5. Pin the extension to your toolbar for easy access

### First Use
1. Click the PhishGuard Pro icon in your toolbar
2. The extension will automatically start analyzing the current webpage
3. Review the analysis results and confidence score
4. Configure settings according to your preferences

## Features

### Real-time Protection
- Automatic analysis of every webpage you visit
- Instant warnings for suspicious sites
- Confidence scores for all analyses

### Manual Scanning
- Click the "Scan Current Page" button in the popup
- Get detailed analysis of any webpage
- View feature explanations and recommendations

### Settings
- Adjust confidence threshold for warnings
- Enable/disable URL or design analysis
- Configure notification preferences
- Privacy and data collection options

## Understanding Results

### Confidence Scores
- **90-100%**: Very likely phishing (Red warning)
- **70-89%**: Suspicious (Yellow warning)
- **50-69%**: Potentially suspicious (Orange warning)
- **0-49%**: Likely legitimate (Green indicator)

### Feature Analysis
- **URL Features**: Domain analysis, suspicious patterns
- **Design Features**: Layout analysis, visual inconsistencies
- **Brand Analysis**: Potential impersonation detection

## Troubleshooting

### Common Issues
- **Extension not working**: Check if it's enabled in Chrome settings
- **False positives**: Adjust confidence threshold in settings
- **Performance issues**: Disable design analysis if needed
